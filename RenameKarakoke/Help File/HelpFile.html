Karaoke Renaming Tool Help File

  1. Importing Help
    • Make sure that the textfile that is imported has only one entry per line
    • The file should be in the format "ID - Artist - Title.kmh"
    • If unformated file names are caught, they will be flagged for user to see
    • Additional songs can be added to the master song list as long as their format is correct
  2. Processing Help
    • Files will be renamed based upon song title
    • If a song has the same title, new files with keep their artist, but ID will be the same to allow users to see duplicates
